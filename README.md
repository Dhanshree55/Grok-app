# 🧠 AI Notes Generator (LLM-Powered Study Assistant)

An AI-powered application that generates **structured, high-quality study notes** from any topic using Large Language Models via the Groq API. Built with a focus on **performance, usability, and real-world application**.

---

## 🎯 Problem Statement

Students often spend significant time converting raw information into structured notes. This project automates that process by transforming a simple topic into **organized, exam-ready content instantly**.

---

## 🚀 Solution

Developed an intelligent notes generation system that:

* Converts user input into **structured prompts**
* Uses **LLM inference (Groq API)** for fast generation
* Outputs **clean, readable notes** with summaries and examples

---

## ✨ Key Features

* ⚡ **Low-latency AI responses** using Groq LLMs
* 🧩 **Well-structured notes**

  * Headings & subheadings
  * Bullet points
  * Real-world examples
  * Summary section
* 🎚 **Custom Modes**

  * Detailed → deep understanding
  * Short → quick revision
* 🎨 **Modern UI/UX**

  * Dark theme + glassmorphism
* ⬇ **Export functionality**

  * Download notes as `.txt`

---

## 🛠 Tech Stack

* **Frontend:** Streamlit
* **Backend:** Python
* **AI Model:** Groq API (LLaMA-based models)
* **Environment Management:** python-dotenv

---

## 🧠 Technical Approach

* Designed **prompt engineering strategy** to ensure:

  * Structured outputs
  * Context relevance
  * Balanced verbosity
* Implemented **LLM API integration** with modular function design
* Built a **responsive UI layer** using Streamlit with custom CSS
* Managed secrets securely using environment variables

---

## 📊 Performance Considerations

* Optimized for **fast inference latency** via Groq
* Lightweight architecture → **low resource usage**
* Scalable design for adding features like chat or PDF export
